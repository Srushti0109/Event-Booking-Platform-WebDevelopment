TechVerse – Event Booking Platform

A modern, responsive event discovery and booking platform built with Next.js 15, TypeScript, and Tailwind CSS.

Users can browse upcoming events, search and filter by category, view detailed event information, and complete event registrations through an intuitive booking flow.
